2.1 安装

安装

1. 安装golang环境

参考 golang 官方文档

2. 使用框架工具

2.1 配置环境变量

export PATH=$PATH:$GOROOT/bin

go env -w GO111MODULE=on
go env -w GOPROXY=https://goproxy.cn,direct

2.2 安装工具

go install gitee.com/csingo/csingo@v0.3.3 // 当前最新版本 v0.3.3

// 查看当前版本
csingo version::show

// 更新工具
csingo version::update

// 初始化工具，安装proto相关工具
csingo tool::init

// 查看帮助信息
csingo

2.3 创建项目

csingo project::create --name=demo

2.4 创建项目相关文件

参考工具帮助信息

csingo help::doc
